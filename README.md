# Grab-A-Bite
***Grab-A-Bite Android Mobile Application***
<p align="justify"> 
An android mobile application that enables the user to calculate his/her recommended daily caloric intake, daily macro-nutrient intake and create a diet schedule. Secondly, it allows the user to check whether a certain packed food item is healthy to consume or not by comparing the macro-nutrient information obtained from the nutrition label image with the set dietary goals.
 </p>

### Background
<p align="justify">
The idea behind the application is motivated by the fact that a lot of individuals want to adopt a healthier eating lifestyle, but problem arises when packed and processed food products comes into the picture. This mainly occurs due to their inability to determine that whether a certain packed food product is healthy to consume or not as per their dietary recommendations. Since packed and processed food items occupy a significant share of the food we buy and consume on regular basis today, therefore this application is expected to be useful in such scenario. We live in a world full of smartphones today and with persistent increase in use of mobile applications that aims at improving quality of life, this application is another such effort which aims at helping individuals to make better dietary choices. It can serve as a handy tool to filter out healthy packed food items with the most common possible use as to help the user choose which items to buy at a grocery or food store.
 </p>

### Tools/Technologies
* Core Java
* Android Studio
* Sqlite
* Google Cloud Vision API

### How to run?
Fork the repository and run via Android Studio.
